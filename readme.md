# uk-summit-guides

A full-stack Django + React guided mountain tours platform with:

- React frontend
- Django REST backend
- seasonal winter/summer theming
- routes discovery
- guided tour bookings
- availability protection
- Stripe payments
- email notifications
- Leaflet / route mapping
- testing and CI/CD

## Project status

In active build.

## Planned architecture

- `frontend/` — Vite + React
- `backend/` — Django REST API
- `docs/` — project notes, board exports, planning docs
